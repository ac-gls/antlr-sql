Module: store/current-user

Current User Store Manages the state for the currently authenticated user using nanostores. Provides reactive atoms for user data, login status, loading state, error handling, and loaded status.

(async, inner) loadCurrentUser() → {Promise.<(CurrentUser|Object)>}

Fetches current user data from the API including related data (dashboards, surveys, permissions, aggregations). Updates loading, error, and loaded state atoms during the request lifecycle.

  • parsers/current-user.js, line 124
Returns:

Current user object with related data, or empty object on error

Example

(inner) resetCurrentUser() → {void}

Resets all current user state to initial values. Clears user data, resets loading/error/loaded flags. Useful for logout scenarios.
